Elastyczność tego rozwiązania jest ograniczona, ale development jest znacznie szbyszy

Najwiekszym plusem tego rozwiązania jest jego szybkość, którą okupujemy ograniczeniem możliwości do tego co oferuje nam API komponentów.

Wiele problemów rozwiązanych przez bibliotekę klasyczny bootstrap jest tutaj niezaimplementowanych i wymaga dodatkowej pracy (np: animacje przy przejściach i klasy przejściowe ".collapsing" etc.).

Niemniej jednak ngBootstrap pozwala na nieinkludowanie zewnętrznych bibliotek znacząco obniżając wagę wynikowych plików js związanych z obsługą aplikacji.
